About Me

Hi, I'm Umer, a passionate Front-End Web Developer with a strong foundation in HTML, CSS, and modern UI frameworks. I'm currently enrolled in the Saylani Mass IT Training Program (SMIT), where I’m building real-world projects and sharpening my development skills through structured, hands-on learning. I specialize in creating clean, responsive, and user-friendly websites. From writing semantic HTML to styling with CSS, I focus on building layouts that not only look great but also perform well across all devices. I'm experienced in using CSS Flexbox and Grid, and I actively apply media queries to ensure full responsiveness. In addition to core HTML and CSS, I work with Bootstrap and Tailwind CSS to speed up development and maintain consistency in design. These frameworks allow me to build scalable, modern interfaces efficiently. I’m also familiar with Git and GitHub, which I use to manage my projects and collaborate with others. My goal is to continue learning, improving, and creating impactful user experiences through the power of front-end development.
